## Recipe for creating a new working project environment
1. Log into Github (https://github.com/login)
2. On the top left side of the page, click the green button labeled "New"
3. Create the repository name and click the green button labeled "Create Repository"
4. Click the drop down arrow on the green button labeled "<> Code"
5. Click the Codespaces tab
6. Open VS Code 
7. On the left bar, click the four boxes icon which will open the extensions menu
8. Search for Codespaces and install the extension
9. In the welcome page, click "Connect to..."
10. In the popup, find "Connect to Codespace..." and click on the Codespace for the repository 
11. Open a bash terminal
12. Create a virtual environment (python3 -m venv venv)
13. Activate the virtual environment (source venv/bin/activate)
14. Create a requirements.txt file with all the packages needed for the project
15. Make sure to save everything
16. Push the changes to Github
